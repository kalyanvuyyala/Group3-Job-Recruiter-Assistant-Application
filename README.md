Job Recruiter Assistant Application (CO7095)

How to run:
1) Open the project folder in PyCharm (job_recruiter_assistant/)
2) Ensure a Python interpreter is configured.
3) Install requirements:
   pip install -r requirements.txt
4) Run the app:
   python -m src.app.main

How to run tests:
- Run all tests:
  pytest -q
- Coverage:
  coverage run -m pytest
  coverage report -m
